Here are five commonly used HTML entities:

 (<) - Represents the less-than sign (<). This entity is used to display the symbol without the browser interpreting it as the opening of an HTML tag.

 (>) - Represents the greater-than sign (>). It's used to display the symbol without the browser interpreting it as the closing of an HTML tag.

 (&) - Represents the ampersand (&). This entity is used to display the ampersand symbol without the browser interpreting it as the beginning of an HTML entity.

 (") - Represents the double quotation mark ("). It's used to display the symbol without the browser considering it as the start or end of an attribute value.

 (©) - Represents the copyright symbol (©). This entity is used to display the copyright symbol in HTML.
