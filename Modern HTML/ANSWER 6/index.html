Differences Between HTML and HTML5

HTML HTML5
HTML stands for HyperText Markup Language. HTML5 is the latest version of HTML introduced in 2014.
Supports limited elements and attributes. Supports a wide range of new elements and attributes such as <header>, <footer>, <video>, <audio>, etc.
Doesn't support native multimedia elements. Provides native support for multimedia elements like audio and video.
Requires browser plugins for rich content. Supports canvas, SVG, and other features for rich content without the need for plugins.
Local storage limited to cookies. Offers support for local storage (localStorage and sessionStorage) for larger amounts of data.
Character encoding mainly based on SGML. Uses UTF-8 as the default character encoding.
